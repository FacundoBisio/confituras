# Palmelita - Maestros Confiteros

Este proyecto consiste en el desarrollo del sitio web oficial para **Palmelita**, una marca de maestros confiteros ubicada en las Islas Canarias. El sitio está diseñado para exhibir su catálogo de mermeladas premium y otros productos artesanales.

## Acerca del Proyecto

Esta página web fue desarrollada por **Facundo Bisio** y ha sido **potenciada con inteligencia artificial** para optimizar el diseño, la estructura y la funcionalidad, asegurando una experiencia de usuario moderna y fluida.

El objetivo principal es ofrecer una presentación elegante de los productos, facilitar el contacto con clientes y distribuidores, y proveer información actualizada a través de una sección de noticias.

## Tecnologías Utilizadas

El desarrollo se ha llevado a cabo utilizando tecnologías web estándar para garantizar compatibilidad, rendimiento y facilidad de mantenimiento:

- **HTML5:** Estructura semántica del sitio.
- **CSS3:** Estilos personalizados, diseño responsivo y efectos visuales modernos (sin uso de frameworks pesados, garantizando una carga rápida).
- **JavaScript:** Interactividad del lado del cliente, incluyendo la galería de imágenes (lightbox) y el selector de idiomas.

## Características Principales

- **Diseño Responsivo:** Adaptado para visualizarse correctamente en dispositivos móviles, tablets y escritorio.
- **Soporte Multi-idioma:** Preparado para alternar entre Español, Inglés, Alemán y Francés.
- **Catálogo de Productos:** Visualización clara y atractiva de las mermeladas con descripciones detalladas.
- **Sección de Noticias:** Espacio dedicado a novedades y actualizaciones de la marca.
- **Formulario de Contacto:** Facilita la comunicación directa con la empresa.
